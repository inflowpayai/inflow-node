# Security Policy

## Reporting a vulnerability

If you believe you've found a security issue in any package in this repo,
please **do not** open a public GitHub issue.

Email `security@inflowpay.ai` with:

- A description of the issue and its impact.
- Steps to reproduce (or a proof of concept).
- The affected package(s) and version(s).
- Any mitigating factors you're aware of.

We'll acknowledge receipt within two business days and aim to provide a
remediation timeline within five.

## Scope

All packages under `packages/` published to npm as `@inflowpayai/*`.
Examples under `examples/` are illustrative and out of scope.
